What is HTTP/2?

HTTP/2 is the lates revision in the HTTP standard. It offers a lot of improvements to page load speed. Read more about HTTP/2 on Wikipedia.

What is HTTP/2 push?

HTTP/2 push is one of the new speed improvemnts from the HTTP/2 revision. It allows servers to push all the resources needed to render a site to a browser in one request. Read more about HTTP/2 on Wikipedia.
